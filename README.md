## 为什么有这个项目

刷题 App 把一套完整的知识网络，**打碎成 683 个互不相关的碎片**让你逐个背。你记住了"A 类功率 ≤25 W"，却不知道它从哪来、为什么 B/C 类不同。于是换个问法就懵、考完即忘、上手仍一头雾水。

我们调研了 GitHub / Gitee / 知乎 / 各 HAM 社区，发现一个**真实的空白**：

| 现存开源资源 | 数量 | 性质 |
|---|---|---|
| 纯题库（JSON/CSV/题库 App） | 数十个 | 只有题目+答案 |
| 带零散注释的题库 | 少量 | 逐题点评、易错点高亮，**非成体系讲解** |
| 个人速成复习笔记（多为 CC-NC） | 若干 | 碎片化、深度浅、二创受限 |
| **成体系、章节化、讲原理、开源可复用的教程** | **❌ 空白** | —— |

**本项目就是来填这个空白的。** 它按官方知识点编码分 5 章，逐节讲透"为什么"，每章末配真题精讲，并以 **CC BY-SA 4.0** 开放许可，欢迎任何人自由学习、分发与二次创作。

## 适合谁

- 正在备考 **A 类操作技术能力验证**，但厌倦了纯刷题、想真正掌握知识的人；
- 想系统了解业余无线电基础（法规 / 通联 / 收发信机 / 天线 / 安全）的爱好者；
- 想在此基础上二次创作（做 B/C 类、做课件、做 Anki、做网页版）的贡献者。

## 如何开始

➡️ **直接从 [`教程/README.md`](./教程/README.md) 进入**，那里有完整的学习路线与考试制度说明。

推荐顺序（先送分、再硬骨头、最后大票仓）：
**导论 → 第 5 章（安全·送分）→ 第 4 章（电工电子·打底）→ 第 3 章（技术核心）→ 第 1 章（法规·最大票仓）→ 第 2 章（通联）+ 反复滚附录 A/B。**

## 内容来源与版本

- **题库**：中国无线电协会（CRAC）发布的 **2025 年 10 月版 A 类题库（683 题）**，经社区仓库结构化整理。题库版权归 CRAC，本仓库为**学习目的**转载整理，详见 [`题库/题库统计.md`](./题库/题库统计.md)。
- **教程讲解**：原创内容，综合《无线电管理条例》《业余无线电台管理办法》（工信部令第 67 号，2024-03-01 施行）等法规原文、通用无线电工程知识与权威科普编写。
- **现行考试制度（2025-10-01 起）**：40 题（单选 32 + 多选 8），40 分钟，答对 ≥30 合格，多选错一即零分。

## ⚠️ 重要声明（请务必阅读）

1. **本教程的讲解部分由 AI（Claude）辅助编写**，作者据官方题库与权威资料整理校订。虽力求准确，**仍可能存在错误**。
2. **以官方文件为准**：若讲解或题目答案与最新官方文件冲突，一律以 **CRAC / 工信部官方文件**为准。考前请到 [CRAC 官网](http://www.crac.org.cn/) 核对最新题库与考试通知（尤其是考试题量、及格线等制度细节）。
3. 本项目仅供**学习交流**，不构成任何官方背书，也不保证考试通过。
