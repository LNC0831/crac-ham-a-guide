# 附录 A · Q 简语与通信缩语速查表

> 对应 B 类教程第 2 章 §2.4《Q 简语与通信缩语》（82 道题）；其中的 DX / 竞赛专用缩语另见 §2.7《DX 通联、网格定位与竞赛》（42 道题）。这是纯记忆内容，建议**打印出来反复滚**。带 ⭐ 的是题库高频考点。
> 用法提示：Q 简语**加问号**就是疑问（"……吗？"），**不加问号**就是陈述（"我……"）。
> B 类说明：Q 简语与通信缩语在 A、B 两类执照中**基本通用**，本附录在 A 类基础上**新增了一组"DX / 竞赛缩语"**（TEST、UP、DN、B4、WKD、SPLIT 等），并补充了 QSX 在分裂/DX 中的用法——这些正是 B 类比 A 类更看重的内容。

---

## 一、Q 简语（Q-code）

> 业余无线电用的是航空/海事 QR-、QS-、QT- 三组中的常用子集。
> ⚠️ **重要：Q 简语是国际电信约定的三字母代码（始于 1912 年），本身没有"字母全称/英文缩写前的全称"**——例如 QSL、QSO、QRZ 并不是某个英文词组的首字母缩写，网上流传的各种"全称"多为后人附会（backronym）。记它的**约定含义**即可，不要去背并不存在的全称。

### QR- 系列（多与"接收/功率/速度/干扰"有关）

| Q 码 | 疑问含义 | 陈述含义 | 备注 |
|---|---|---|---|
| QRA | 你的电台名称是？ | 我的电台名称是…… | |
| QRG | 我的准确频率是？ | 你的频率是…… | |
| QRH | 我的频率变动吗？ | 你的频率在漂移 | |
| QRK | 我的信号可懂度如何？ | 你的可懂度为……（1–5） | RST 中的 R 即源于此 |
| ⭐QRL | 频率被占用吗？/你忙吗？ | 频率正被占用／我正忙 | 守听询问常用 |
| ⭐QRM | 受到（他台）干扰吗？ | 受到**人为干扰** | **M=Man 人为** |
| ⭐QRN | 受到天电干扰吗？ | 受到**自然（天电）干扰** | **N=Nature 自然** |
| ⭐QRO | 要增大功率吗？ | 增大发射功率 | **O=大** |
| ⭐QRP | 要减小功率吗？ | 减小功率／小功率操作（<5 W） | **P=小**；呼号后缀 /QRP 即此意 |
| ⭐QRQ | 要发快些吗？ | 加快发报速度 | Q=Quick |
| ⭐QRS | 要发慢些吗？ | 放慢发报速度 | S=Slow |
| ⭐QRT | 要我停止发射吗？ | 我要**关机**了 | |
| ⭐QRU | 你有消息给我吗？ | 我这里**没有消息** | |
| ⭐QRV | 你准备好了吗？ | 我**准备好**了（就绪） | |
| QRX | 你何时再呼叫我？ | 请**稍等**／我将在……再呼叫 | |
| ⭐QRZ | 谁在呼叫我？ | 呼叫你的是…… | 没听清呼号时用；DX/竞赛结束呼叫时常发 |

### QS- 系列（多与"信号质量/联络动作"有关）

| Q 码 | 疑问含义 | 陈述含义 | 备注 |
|---|---|---|---|
| ⭐QSA | 我的信号强度如何？ | 你的强度为……（**1–5 级**） | 早期信号报告法 |
| ⭐QSB | 我的信号衰落吗？ | 你的信号正在**衰落** | |
| QSD | 我的键控有缺陷吗？ | 你的键控有缺陷（手法/键击声等） | CW 用 |
| QSK | 你能在发报间隙听到我吗？ | 我能在发报**间隙听到插入**（全双工键控） | |
| ⭐QSL | 你能确认收妥吗？ | 我**确认收妥**了你的消息（也指 QSL 卡） | |
| ⭐QSO | 你能通联……吗？ | （一次）**通联** | |
| ⭐QSP | 你能转信给……吗？ | 我可以**转信**给某台 | |
| QSV | 要我发一串 V 调试吗？ | 我发一串 V 供你调整 | |
| ⭐QSX | 你在……守听吗？ | 我在（某频率）**守听** | **DX/分裂关键**：见下文 §四 |
| ⭐QSY | 要改到……频率吗？ | 我要**改变频率**到…… | |

### QT- 系列

| Q 码 | 含义 |
|---|---|
| ⭐QTH | 位置（"我的 QTH 是成都"） |
| QTR | 准确时间 |

> **最易混的三组，务必背死**：
> - **QRM（人为）/ QRN（自然）**
> - **QRO（大）/ QRP（小）/ QRQ（快）/ QRS（慢）**
> - **QRT（关机）/ QRU（无消息）/ QRV（就绪）/ QRZ（谁在叫我）**

## 三、DX / 竞赛专用缩语（B 类重点，对应 §2.7）

> B 类执照更强调 DX 追逐与竞赛操作，下面这组缩语在普通本地通联中较少出现，但在 DX 围猎和比赛中**几乎每条信息都在用**，务必背熟。
> 三列仍保持"缩语 | 英文源词/词源 | 中文"格式；约定型记号（非英文缩写的）已据实标注。

| 缩语 | 英文源词 / 词源 | 中文 |
|---|---|---|
| ⭐DX | Distance / Distant | 远距离台／DX 台（"CQ DX"＝呼叫所有远方电台） |
| ⭐TEST | contest（截短，"正在竞赛中"） | 我正在参加比赛（如 "B1CRA TEST"＝B1CRA 正在参赛） |
| ⭐CONTEST | Contest | 竞赛 |
| ⭐UP | Up | 分裂操作：请在比我发射频率**更高**处呼叫（"UP 5"＝高 5 kHz 守听） |
| ⭐DN / DOWN | Down | 分裂操作：请在比我发射频率**更低**处呼叫 |
| ⭐SPLIT | Split（分裂） | **分裂操作**：发射与接收用不同频率（稀有台为疏导堆叠时常用） |
| ⭐QSX | Q 简语（约定码，无字母全称），陈述义＝我在某频率守听 | DX/分裂中表示"**我在 ××× 守听你的回叫**"（例：QSX UP 5 / QSX 7074 KHZ） |
| ⭐B4 | before（数字 4 谐音 "for/fore"，整体读作 before） | **之前已通联过**（竞赛中抄到，提示查日志是否重复） |
| ⭐WKD | worked | **已通联**（与 B4 同义，提醒该台此前已联过） |
| AGN | Again | 再来一次/重发（竞赛中请对方重报交换信息） |
| QRZ | Q 简语，疑问义＝谁在呼叫我 | 结束一次比赛 QSO、招呼下一个呼叫者时常发"QRZ?" |
| CQ DX | CQ + DX | 呼叫所有远方（境外）电台 |
| NR | Number | （竞赛）序号／交换号 |
| GRID | Grid（梅登海德网格） | 网格定位（如 OM89，VHF/UHF 竞赛常作交换信息） |
| 599 / 5NN | RST（CW 比赛中 9 常用 N 代发以提速） | 标准信号报告 599（5NN 即 599） |

> **分裂三件套（最易混，背死）**：
> - **UP（向上）/ DN·DOWN（向下）**——告诉别人去哪儿呼叫**你**；
> - **SPLIT**——总称"收发异频"这件事本身；
> - **QSX ×××**——稀有台**主动报出**自己在哪个频率守听（"我在这儿听你"）。
> 例："VK0XX UP 5"＝VK0XX 在自己发射频率**高 5 kHz**处守听，回叫请去那里。
>
> **重复联络两件套**：**B4 / WKD** 都表示"这台**之前联过了**"——竞赛抄到要回头查日志，别白联一次（重复联络不计分）。

## 四、一次典型 CW 通联示例（把缩语串起来）

```
CQ CQ CQ DE BH1ZZZ BH1ZZZ K          （BH1ZZZ 呼叫，请任意台回答）
BH1ZZZ DE BH8YYY BH8YYY KN           （BH8YYY 应答，只请 BH1ZZZ 回）
BH8YYY DE BH1ZZZ = GM ES TNX FER CALL（早上好，谢谢呼叫）
= UR RST 599 599 = MY QTH IS BEIJING = OP IS XIAO MING = HW? KN
                                      （给你 599，我在北京，我叫小明，怎么样？）
…
= TU FER QSO = 73 ES GL = BH8YYY DE BH1ZZZ SK    （谢谢通联，73 祝好，结束）
```

> 看懂这段，你就掌握了 §2.4 大部分缩语的实际用法。

## 五、一次 DX / 竞赛分裂通联示例（B 类专属）

```
CQ DX CQ DX DE VK0XX VK0XX UP 5 K        （稀有台 VK0XX 呼叫 DX，分裂：请在高 5 kHz 呼叫）
（你把电台设成分裂：接收守听 VK0XX 的发射频率，发射移到高 5 kHz）
BH1ZZZ BH1ZZZ                            （你在高 5 kHz 处快速报出自己呼号）
BH1ZZZ DE VK0XX 599 5NN                  （VK0XX 回你，给 599）
599 TU                                   （你回 599 致谢，一次 DX QSO 完成）
VK0XX QRZ? UP                            （VK0XX 招呼下一个呼叫者，仍分裂向上）
```

竞赛片段示例：

```
B1CRA TEST UP 5      （B1CRA 正在参赛，接收频率从发射频率向上偏移 5 kHz）
…（你回叫并交换信息）…
B4                   （对方抄到后提示：你我之前已联过——回头查日志，别重复）
```

> 看懂这两段，你就掌握了 §2.7 中 DX 与竞赛操作的核心缩语：**CQ DX / TEST / UP / DN / SPLIT / QSX / B4 / WKD**。
