# 附录 A · Q 简语与通信缩语速查表

> 对应第 2 章 §2.4（82 道题）。这是纯记忆内容，建议**打印出来反复滚**。带 ⭐ 的是题库高频考点。
> 用法提示：Q 简语**加问号**就是疑问（"……吗？"），**不加问号**就是陈述（"我……"）。

---

## 一、Q 简语（Q-code）

> 业余无线电用的是航空/海事 QR-、QS-、QT- 三组中的常用子集。

### QR- 系列（多与"接收/功率/速度/干扰"有关）

| Q 码 | 疑问含义 | 陈述含义 | 备注 |
|---|---|---|---|
| QRA | 你的电台名称是？ | 我的电台名称是…… | |
| QRG | 我的准确频率是？ | 你的频率是…… | |
| QRH | 我的频率变动吗？ | 你的频率在漂移 | |
| QRK | 我的信号可懂度如何？ | 你的可懂度为……（1–5） | |
| ⭐QRL | 频率被占用吗？/你忙吗？ | 频率正被占用／我正忙 | 守听询问常用 |
| ⭐QRM | 受到（他台）干扰吗？ | 受到**人为干扰** | **M=Man 人为** |
| ⭐QRN | 受到天电干扰吗？ | 受到**自然（天电）干扰** | **N=Nature 自然** |
| ⭐QRO | 要增大功率吗？ | 增大发射功率 | **O=大** |
| ⭐QRP | 要减小功率吗？ | 减小功率／小功率操作（<5 W） | **P=小** |
| ⭐QRQ | 要发快些吗？ | 加快发报速度 | Q=Quick |
| ⭐QRS | 要发慢些吗？ | 放慢发报速度 | S=Slow |
| ⭐QRT | 要我停止发射吗？ | 我要**关机**了 | |
| ⭐QRU | 你有消息给我吗？ | 我这里**没有消息** | |
| ⭐QRV | 你准备好了吗？ | 我**准备好**了（就绪） | |
| QRX | 你何时再呼叫我？ | 请**稍等**／我将在……再呼叫 | |
| ⭐QRZ | 谁在呼叫我？ | 呼叫你的是…… | 没听清呼号时用 |

### QS- 系列（多与"信号质量/联络动作"有关）

| Q 码 | 疑问含义 | 陈述含义 | 备注 |
|---|---|---|---|
| ⭐QSA | 我的信号强度如何？ | 你的强度为……（**1–5 级**） | 早期信号报告法 |
| ⭐QSB | 我的信号衰落吗？ | 你的信号正在**衰落** | |
| QSD | 我的键控有缺陷吗？ | 你的键控有缺陷（手法/键击声等） | CW 用 |
| QSK | 你能在发报间隙听到我吗？ | 我能在发报**间隙听到插入**（全双工键控） | |
| ⭐QSL | 你能确认收妥吗？ | 我**确认收妥**了你的消息（也指 QSL 卡） | |
| ⭐QSO | 你能通联……吗？ | （一次）**通联** | |
| ⭐QSP | 你能转信给……吗？ | 我可以**转信**给某台 | |
| QSV | 要我发一串 V 调试吗？ | 我发一串 V 供你调整 | |
| ⭐QSX | 你在……守听吗？ | 我在（某频率）**守听** | |
| ⭐QSY | 要改到……频率吗？ | 我要**改变频率**到…… | |

### QT- 系列

| Q 码 | 含义 |
|---|---|
| ⭐QTH | 位置（"我的 QTH 是成都"） |
| QTR | 准确时间 |

> **最易混的三组，务必背死**：
> - **QRM（人为）/ QRN（自然）**
> - **QRO（大）/ QRP（小）/ QRQ（快）/ QRS（慢）**
> - **QRT（关机）/ QRU（无消息）/ QRV（就绪）/ QRZ（谁在叫我）**

## 二、通信缩语（多用于 CW / 数字模式）

### 问候与祝愿

| 缩语 | 含义 | 缩语 | 含义 |
|---|---|---|---|
| ⭐GM | 早上好 (Good Morning) | ⭐GA | 下午好 (Good Afternoon) |
| ⭐GE | 晚上好 (Good Evening) | ⭐GN | 晚安 (Good Night) |
| ⭐**73** | 致意/问候（最常用） | 88 | 吻/爱意（多对 YL） |
| ⭐GL | 祝好运 (Good Luck) | GB | 再见 (Good Bye) |
| ⭐HNY | 新年快乐 | CUL / CU AGN | 回头见 |

### 礼貌与确认

| 缩语 | 含义 | 缩语 | 含义 |
|---|---|---|---|
| ⭐PSE / PLS | 请 (Please) | ⭐TNX / TKS | 谢谢 (Thanks) |
| ⭐TU | 谢谢你 (Thank You) | MNI/MNY TNX | 非常感谢 |
| ⭐SRI | 抱歉 (Sorry) | ⭐R | Roger（收到/确认） |
| ⭐FB | 太棒了 (Fine Business) | GLD | 高兴 (Glad) |
| HPE | 希望 (Hope) | HPY/HPI | 幸福 (Happy) |

### 常用词与流程

| 缩语 | 含义 | 缩语 | 含义 |
|---|---|---|---|
| ⭐UR | 你的/你是 (Your) | ⭐ES | 和 (And) |
| ⭐FER | 为了/对于 (For) | ⭐NW | 现在 (Now) |
| ⭐AGN | 再一次 (Again) | WL | 将要 (Will) |
| ⭐FREQ | 频率 | TEMP | 温度 |
| ⭐CQ | 普遍呼叫（找任意台） | ⭐DX | 远方/远距离 |
| ⭐K | 请任意台回答（完成发送、守听） | ⭐KN | 只请**对方**回答 |
| DE | 来自/这里是 (from) | AHR | 另一个 (another) |
| CK | 检查 (Check) | WX | 天气 |
| RIG | 电台设备 | PWR | 功率 |
| HST | 高速收发报 | ARDF | 业余无线电测向 |

### 人物称呼

| 缩语 | 含义 |
|---|---|
| ⭐OM | "老朋友/老伙计"（对男操作员的昵称，**无关年龄**） |
| ⭐YL | 女士 (Young Lady) |
| ⭐XYL | 妻子 |
| ⭐OP | 操作员 (Operator) |

### 时间与邮寄/卡片

| 缩语 | 含义 |
|---|---|
| GMT | 格林尼治时间（≈ UTC） |
| ⭐SAE | 写好收信人地址的信封 |
| ⭐SASE | 写好地址**且贴好邮票**的信封 |
| ⭐BURO | QSL 卡片管理局 |
| BOX / P O BOX | 邮政信箱 |
| IRC | 国际回邮券 |
| SWL | 短波收听者（只收听不发射，会寄收听卡） |
| ADR / ADDR | 地址 |

| 缩写 | 天线 |
|---|---|
| ⭐DP | 偶极天线 (Dipole) |
| ⭐GP | 垂直接地天线 (Ground Plane) |
| ⭐VER | 垂直天线 (Vertical) |
| ⭐LW | 长线天线 (Long Wire) |
| ⭐YAGI | 八木天线 |
| ⭐BEAM | 定向天线 |

## 三、一次典型 CW 通联示例（把缩语串起来）

```
CQ CQ CQ DE BH1ZZZ BH1ZZZ K          （BH1ZZZ 呼叫，请任意台回答）
BH1ZZZ DE BH8YYY BH8YYY KN           （BH8YYY 应答，只请 BH1ZZZ 回）
BH8YYY DE BH1ZZZ = GM ES TNX FER CALL（早上好，谢谢呼叫）
= UR RST 599 599 = MY QTH IS BEIJING = OP IS XIAO MING = HW? KN
                                      （给你 599，我在北京，我叫小明，怎么样？）
…
= TU FER QSO = 73 ES GL = BH8YYY DE BH1ZZZ SK    （谢谢通联，73 祝好，结束）
```

> 看懂这段，你就掌握了 §2.4 大部分缩语的实际用法。
