## 2.1 ITU 三大分区与字母解释法

### 2.1.1 ITU 把世界分成 3 个区，中国在第 3 区

为了全球协调频率，**国际电信联盟（ITU）的《无线电规则》把世界分为 3 个无线电管理区域**：

- **第 1 区**：欧洲、非洲、中东、俄罗斯的亚洲部分、蒙古及部分西北亚国家；
- **第 2 区**：南北美洲；
- **第 3 区**：亚洲（除俄罗斯、蒙古和部分西北亚国家）和大洋洲——**中国在第 3 区**。

为什么要分区？因为同一个业余频段在不同区的边界可能不同（例如 40 米段：第 3 区是 7.0–7.2 MHz，第 2 区是 7.0–7.3 MHz）。记住"**中国＝第 3 区**"，以及"分区是 ITU 用《无线电规则》定的"。

### 2.1.2 字母解释法（语音字母表）——为什么必须用它

无线电信道有噪声，"B、D、P、T、V"这些字母发音相近，极易听错。所以全世界统一用 **ITU/NATO 语音字母表**把每个字母念成一个不会混淆的单词：

> A=Alfa, B=**Bravo**, C=**Charlie**, H=**Hotel**, X=**Xray**, Y=**Yankee**, Z=**Zulu** ……（完整 26 字母表见 [附录 B](./附录B-常用对照表.md)）

例：呼号 **BH1XYZ** 念作 **"Bravo Hotel One Xray Yankee Zulu"**。考试会给一个呼号，让你选正确的拼读——只要背熟字母表就能秒选。**数字照英文读**（One/Two/…），呼号里的斜杠 `/` 念 **"Portable"**（如 K3TRX/VE6 = …Xray **Portable** Victor Echo Six）。

### 2.1.3 边缘频率的载波设置（结合 §1.7.3 一起理解）

2.1 里有一类"计算题"：给定一个细分频段（如 144.035–145.8 MHz）、调制方式和发射带宽，问主载波能设在什么范围。原理就是[第 1 章讲的"信号不能越界"](./01-法律法规与频率管理.md#173-实际可用发信频率为什么不能贴着边发)：

- **FM**（带宽<20 kHz，约 ±10 kHz）：载波要离两端各缩进约半个带宽 + 频率容限。例：144.035 + 0.012 ≈ **144.047**；145.8 − 0.012 ≈ **145.788**。
- **USB**（带宽<3 kHz）：载波（抑制载波点）设在边带的一端，缩进量更小（约 2 kHz）。

考试不必心算精确，记住**"载波要从频段边缘往里缩进，缩进量≈半带宽（FM）或一个边带宽（SSB）再加频率容限"**，对照选项选那个"两端都往里缩"的即可。同理，**144–144.035、145.8–146、435–438** 这些**留给卫星业余业务**的子段，本地话音不要占用。

## 2.2 通联程序与通信日志

### 2.2.1 标准通联流程

一次完整的通联（QSO）大致是：

```
1. 守听 → 确认频率空闲（"这个频率有人用吗？"）
2. 发起呼叫 CQ：  "CQ CQ CQ, this is BH1ZZZ（拼读）, calling CQ and standing by"
3. 对方应答：     先报【对方呼号】，再报【自己呼号】  ← 顺序很重要！
4. 交换【信号报告 RST】 ← 建立通信后第一件要交换的事
5. 交换 QTH（位置）、姓名、设备、天气等
6. 互道 73，报全呼号，结束
```

**两个高频考点**：
- **应答 CQ 时**：先报**对方**呼号，再报**自己**呼号（"BH8YYY, this is BH1ZZZ"）。顺序反了就错。
- **建立通信后第一件事是交换信号报告**——因为双方需要它来**了解传播情况、安排后续操作**，不是为了"校准功率""决定是否放弃"。

### 2.2.2 RST 信号报告制——理解三个字母

RST 是给对方信号打分的标准格式，三个字母含义务必分清：

| 字母 | 含义 | 范围 | 用于 |
|---|---|---|---|
| **R**（Readability） | **可辨度**（能不能听清内容） | **1–5** | 所有方式 |
| **S**（Signal Strength） | **信号强度**（表头读数 / 听感强弱） | **1–9** | 所有方式 |
| **T**（Tone） | **音调纯度**（CW 信号音是否纯净） | **1–9** | **仅 CW** |

所以：
- **话音（SSB/FM）只报两位 RS**，如 "59"（完全可读+极强）、"57"。
- **CW 报三位 RST**，如 "599"。

> 📝 **经典真题**：CW 和 FM 都收到信号强度为 5、听感清晰无失真，应分别给报告 **559（CW）和 55（FM）**。因为 CW 要带音调位 T（纯音=9），FM 没有 T 位只报 RS。选"都给 599/59"是错的——不能不看强度表瞎报满分。

### 2.2.3 插入与应急礼仪

- **请求插入两台之间的对话**：在一方**刚停止讲话**的间隙发 "break / 插入"，得到回应后说明 "BH1ZZZ 请求插入"，待**正式邀请**后再加入。不要在别人讲话时硬插、不要加大功率反复尝试。插入前还应**先想想自己的加入会不会破坏原双方的交流乐趣**。
- **听到遇险电台要求插入**：**立即与该台确认险情、了解需要什么帮助**——应急通信优先级最高，一切让路。
- **遇到不规范操作**：若只是技巧不规范、未造成严重干扰，应**私下**（电话/邮件）善意提建议，**不要在频率上当众指责**。
- **遇到违法/不明电台严重干扰**：**不予理睬，收集信息后向无线电管理机构举报**——不要在频率上对骂、不要用大功率压制（压制本身就违法）。

### 2.2.4 中继台与频差

通过中继台呼叫，方法和直接通联一样：**先报对方呼号、再报自己呼号**。要事先设好收发频差（offset）：

| 频段 | 标准频差 |
|---|---|
| **144 MHz** | **0.6 MHz**（600 kHz） |
| **430 MHz** | **5 MHz** |

中继台使用原则：**除必要简短联络外，上下行频率应保持空闲**，以备随时响应应急呼叫；不能让它长期占满、不能拿来发协会通知或学生交流学习。

### 2.2.5 QSL 卡片与通信日志

- **通信日志（Log）必记内容**：**通信时间、频率、模式、对方呼号、双方信号报告**（英文字段 DATE / TIME / FREQ / MODE / CALL / RST）。注意"对方姓名/天气/设备"不是**必需**项。
- **QSL 卡片**：通联的书面确认卡，用于确认联络、申请奖状、收藏交流。卡上至少要有：**双方呼号、信号报告、联络时间、模式、频率、操作员签章与通信地址**。
- 几个术语（详见 [附录 A](./附录A-Q简语与缩语速查表.md)）：
  - **SWL** = 短波收听者（只收听、不发射，会寄"收听卡"，收到后应回卡致意）；
  - **IRC** = 国际回邮券、**SAE** = 写好地址的信封、**SASE** = 写好地址且贴好邮票的信封、**BURO** = QSL 卡片管理局。
  - 不应赠送/散发**空白** QSL 卡；没真正联络过不应发确认卡；漏填错填对方呼号不能拿对方名字顶替。

## 2.3 频率的守听与礼让

核心就一条：**发射前先守听，确认频率空闲。** 哪怕一时没听到，也可能是只听到了通联中的一方，所以正式 CQ 前应先问一句 **"这个频率有人用吗？（Is this frequency in use?）"**，确认无人应答再呼。

- 表达"频率是否占用"的正确英文："Is this frequency in use?" / "Is anyone on this frequency?" / "Anybody here?"。而 "Calling you, Roger?" **不是**询问频率占用的句子（它是错误选项）。
- 报位置用 **QTH**（"My QTH is Chengdu"）；注意别和 QRT（关机）、QRO（增大功率）、QRV（准备好了）混。
- 听到 "Cheerio / Ciao ciao / Do svidaniya" 这类，是不同国家爱好者**互道再见、致以祝愿**。

## 2.4 Q 简语与通信缩语（82 题的大头）

> 这一节量最大，但有规律。**Q 简语**是三字母代码（Q 开头），既可作陈述也可加"？"作疑问；**通信缩语**是英文单词的省音写法（多用于 CW/数字模式）。**完整速查表见 [附录 A](./附录A-Q简语与缩语速查表.md)**，这里讲"怎么成组记、怎么不混"。

### 2.4.1 Q 简语按"功能场景"记

把 Q 码按使用场景分组，比按字母背好记得多：

**① 功率与速度（QR- 系列里管"快慢大小"的）**
| Q 码 | 陈述含义 | 记忆 |
|---|---|---|
| **QRO** | 增大发射功率 | O=大（big O） |
| **QRP** | 减小发射功率 / 小功率操作（<5 W） | P=小（puny） |
| **QRQ** | 加快发报速度 | Q=Quick |
| **QRS** | 放慢发报速度 | S=Slow |

**② 状态与意图**
| Q 码 | 含义 |
|---|---|
| **QRT** | 我要关机了（停止发射） |
| **QRV** | 我准备好了（ready） |
| **QRU** | 我这里没有消息了（nothing for you） |
| **QRL** | 频率正被占用 / 我正忙 |
| **QRZ** | 谁在呼叫我？ |

**③ 干扰与信号质量**
| Q 码 | 含义 |
|---|---|
| **QRM** | 受到**人为**干扰（其他电台） |
| **QRN** | 受到**自然**干扰（天电、雷电噪声） |
| **QSB** | 信号在**衰落**（强弱起伏） |
| **QSA** | 信号强度（早期用 1–5 级表示） |
| **QSD** | 你的键控有缺陷（CW 手法/键击声等） |
| **QSK** | 我能在发报间隙听到插入（全双工键控） |

**④ 联络动作**
| Q 码 | 含义 |
|---|---|
| **QSO** | （一次）通联 |
| **QSL** | 我确认收妥了你的消息（也指确认卡） |
| **QSP** | 我可以帮你**转信**给某台 |
| **QSX** | 我在某频率**守听** |
| **QSY** | 我要**改变频率**到…… |
| **QTH** | 位置 |

**最易混的三组，务必分清**：
- **QRM（人为干扰）vs QRN（自然干扰）**——M=Man（人），N=Nature（自然）。
- **QRO（大）vs QRP（小）vs QRQ（快）vs QRS（慢）**。
- **QRT（关机）vs QRV（就绪）vs QRU（无消息）vs QRZ（谁在叫我）**。

### 2.4.2 通信缩语：英文省音，见词猜意

缩语多是把英文单词去掉元音。掌握几个高频的，其余见 [附录 A](./附录A-Q简语与缩语速查表.md)：

- **问候/祝愿**：GM=早上好、GA=下午好、GE=晚上好、GN=晚安；**73**=致意（最常用！）、GL=祝好运、HNY=新年快乐。
- **礼貌用语**：PSE/PLS=请、TNX/TKS=谢谢、TU=谢谢你、SRI=抱歉、R=Roger（收到/确认）、FB=太棒了。
- **常用词**：UR=你的、ES=和、FER=为了、NW=现在、AGN=再一次、HPE=希望、WL=将要、FREQ=频率、TEMP=温度。
- **人物**：OM=老朋友（男，无关年龄）、YL=女士、XYL=妻子、OP=操作员。
- **流程符号**：CQ=普遍呼叫、DX=远方/远距离、K=请任意台回答（完成发送守听）、KN=只请对方回答、DE=来自。
- **天线缩写**：DP=偶极天线、GP=垂直接地天线、VER=垂直天线、LW=长线天线、YAGI=八木、BEAM=定向天线、EL=振子单元、ANT=天线、ATU=天调。

### 📝 2.4 真题精讲

**真题（单选）** QRP 操作时对方反映信号不好，追问是否需要增大功率，可发送：✅ A. **QRO？**
> QRP（小功率）→ 想加大 → QRO（大功率）。O 大 P 小，成对记。

**真题（单选）** 称对方操作员为 "old man"，缩语是：✅ A. **OM**
> OM=老朋友/老伙计，**无关年龄、无关性别尊称**；YL=女士，XYL=妻子。这组是固定考点。

## 2.5 发射类别（emission designator）与调制方式

### 2.5.1 发射类别符号：会"拆解"就不用背

国际上用**三个符号**标示一种发射的特性，**学会拆解结构，任何符号都能读懂**：

```
      F 3 E
      │ │ └─ 第3位：被传信息类型
      │ └─── 第2位：调制信号的性质
      └───── 第1位：主载波调制方式
```

| 第1位（载波调制） | 第2位（调制信号） | 第3位（信息类型） |
|---|---|---|
| **A** = 双边带调幅（AM） | **0** = 无调制 | **A** = 人工电报（莫尔斯，听抄） |
| **J** = 单边带抑制载波（SSB） | **1** = 数字、无副载波 | **B** = 自动电报（机器收，如 RTTY） |
| **F** = 调频（FM） | **2** = 数字、有副载波 | **E** = 电话（语音） |
| **G** = 调相（PM） | **3** = 模拟（单通道） | **F** = 电视/图像 |

对照拆解高频符号：

| 符号 | 拆解 | 实际含义 |
|---|---|---|
| **A1A** | 调幅 + 数字无副载波 + 人工电报 | **CW（莫尔斯电报）** |
| **J3E** | 单边带 + 模拟 + 电话 | **SSB 单边带话** |
| **F3E** | 调频 + 模拟 + 电话 | **FM 调频话** |
| **F3F** | 调频 + 模拟 + 电视 | **FM 图像（ATV/SSTV）** |
| **F2B** | 调频 + 数字有副载波 + 自动电报 | FM 上的数据/RTTY |

**记住四个最常考的：A1A=CW、J3E=SSB话、F3E=FM话、F3F=FM图像。** 其余按表拆解。

### 2.5.2 三种基本调制：AM / FM / PM 与 CW

- **AM（调幅）**：载波**幅度**随话音变化。**SSB 是 AM 的一种**（去掉载波和一个边带的调幅）。
- **FM（调频）**：载波**频率**随话音变化。
- **PM（调相）**：载波**相位**随话音变化。
- **CW（等幅电报）**：通断载波发莫尔斯码，是**带宽最窄**的方式。

### 2.5.3 怎么选调制方式（理解"为什么"）

| 场景 | 选用 | 原因 |
|---|---|---|
| **VHF/UHF 本地通联** | **FM** | 抗噪好、音质佳，适合近距离强信号 |
| **VHF/UHF 远距离/弱信号** | **SSB** | 带宽窄、能量集中，弱信号也能听清 |
| **HF 短波语音** | **SSB** | 带宽窄、效率高，适合远距离 |
| **最远距离/弱信号极限** | **CW** | 带宽最窄，等功率下传得最远 |

**FM 与 SSB 的优缺点对比**（高频考点）：
- **SSB 相比 FM 的优点**：**发射带宽窄**（约 3 kHz vs FM 的 10–16 kHz），频谱利用率高、弱信号能力强。
- **FM 相比 SSB 的缺点**：带宽宽、**无法同时守听多个信号**（一个强信号会"捕获"接收机）。
- **不用 FM 发莫尔斯码**：因为 FM 带宽远超传莫尔斯码的必要带宽，违反"带宽保持在必要最低值"的规定。

### 2.5.4 边带选择的"习惯约定"

SSB 用上边带（USB）还是下边带（LSB），是**全球爱好者沿用至今的习惯**，不是物理必然：

- **10.1 MHz 以下**（如 7 MHz、3.5 MHz）：用 **LSB（下边带）**；
- **10.1 MHz 以上**（含 14 MHz 及以上 HF、以及 **VHF/UHF**）：用 **USB（上边带）**。

记忆：**"低频下、高频上"**；VHF/UHF 归入"高频"用 USB。这纯属约定俗成（选项里"更利于传播""能同时收双边带"都是错的）。

另外：**发起 SSB 呼叫时，与相邻电台至少留 2–3 kHz 间隔**（SSB 信号约占 2.4–3 kHz）；**给话筒灌入过强音频会过调制**，产生谐波和互调、展宽带宽、质量下降（不会"通信更远"）。

## 2.6 数字通信模式（了解即可，5 题）

近年新增的数字语音/数据模式，A 类只需认得名字和基本特征：

| 模式 | 关键特征 |
|---|---|
| **YSF**（Yaesu System Fusion） | 八重洲数字语音/数据融合，调制方式 **C4FM**，可经中继转发 |
| **D-STAR**（Icom） | 数字语音/数据，可经中继和**互联网/热点**扩展范围，支持 GPS 定位报告 |
| **FreeDV** | **开源**数字语音，带宽窄、适合 HF，可用于研究和自制装备 |
| **DATV** | 数字业余电视，卫星转发常用 **DVB-S2** 协议（QPSK/8PSK/16APSK/32APSK） |
| **DSSTV**（DRM 协议） | 数字慢扫描电视，倾向用 **SSB**（保留基带特性、利于衰落信道纠错） |

- **中继频差**：144→0.6 MHz，430→5 MHz。
- **日志必记**：时间、频率、模式、对方呼号、双方 RST。
- **Q 码三组易混**：QRM 人为 / QRN 自然；QRO 大 / QRP 小 / QRQ 快 / QRS 慢；QRT 关机 / QRV 就绪 / QRU 无消息 / QRZ 谁叫我。
- **发射类别**：第1位载波调制(A/J/F/G)、第2位信号性质(0/1/2/3)、第3位信息类型(A/B/E/F)。A1A=CW、J3E=SSB话、F3E=FM话、F3F=FM图像。
- **调制选择**：本地 FM、弱信号/远距 SSB、极限 CW；SSB 比 FM 带宽窄。
- **边带约定**：10.1 MHz 以下 LSB，以上（含 V/U）USB。
