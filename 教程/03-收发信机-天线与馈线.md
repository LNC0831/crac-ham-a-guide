# 第 3 章 · 收发信机、天线与馈线

> **本章在 683 题中占 194 题（28%），是技术核心。**
>

>
> **话音 →〔发射机：调制+放大〕→〔馈线〕→〔天线：电流变电波〕→ 空间传播 →〔天线：电波变电流〕→〔馈线〕→〔接收机：放大+解调〕→ 话音。**

## 3.1 收发信机与操作面板

### 3.1.1 最小系统：一台电台至少要有什么

- **收发信机（transceiver）** = 接收机 + 发射机的合体。
- **能工作的接收系统**最少要有：**接收天线 + 解调器**（把电波变回信息）。
- **能工作的发射系统**最少要有：**射频振荡器 + 调制器 + 发射天线**（产生载波、加载信息、发射出去）。
- 直流电源线要**又粗又短**——目的是**减小发射时电源线上的压降**（大电流时细长线压降大，会让功率不足、信号变差）。

### 3.1.2 单工 / 双工 / 半双工（按"能否同时双向"判断）

| 方式 | 定义 | 典型例子 |
|---|---|---|
| **单工**（simplex） | 同一通路上**交替**单向 | 两台靠 **PTT** 切换收发的对讲机互通 |
| **双工**（duplex） | 两个方向**同时**传输 | 卫星通信（地面台 430 收 + 144 发**同时**进行） |
| **半双工**（semi-duplex） | 一端单工、一端双工 | **中继台**（双工）与对讲机（单工 PTT）之间 |

判断窍门：**能不能同时收发**？能 = 双工；靠 PTT 来回切 = 单工；一端能同时、一端要切 = 半双工。

### 3.1.3 面板缩写速记（按"收/发"分两类）

操作面板上的英文缩写，本质都在调"收信"或"发信"的某个环节。**分类记**最不容易混：

**接收侧（让你听得更好）**
| 缩写 | 全/含义 | 作用 |
|---|---|---|
| **VOL** | Volume | 音量 |
| **SQL** | Squelch（静噪） | 无信号时关掉刺耳噪声；阈值越高越"安静"，**收弱 FM 信号要关静噪/调 0** |
| **AGC** | 自动增益控制 | 检测中频电平、自动调增益**防过载**；收 SSB/AM 用长时间常数、收 FM/数据用短时间常数；**弱信号+强干扰同时出现时应关 AGC** |
| **ATT** | Attenuator 衰减器 | 强信号时接入，**防前级过载/互调**（AGC 之外的补充） |
| **PRE** | Preamplifier 前置放大 | 收**微弱**信号时接入，降低后级噪声影响 |
| **NB** | Noise Blanker 抑噪 | 切除**脉冲噪声**（如汽车火花塞干扰），与 SQL 不同 |
| **ATT/AGC/NB** 区别 | —— | ATT 衰减大信号、NB 切脉冲、SQL 静音、AGC 稳增益 |

**发射侧（让你发得更规范）**
| 缩写 | 含义 | 作用 |
|---|---|---|
| **PTT** | Push To Talk | 按键发射（接地接通时转发射） |
| **VOX** | 声控 | 检测话筒有无语音，**自动切换收发**（"免提"，腾出双手） |
| **ALC** | 自动电平控制 | 检测射频输出、防**过驱动**导致失真/故障 |
| **PROC** | Processor 语音压缩 | 压强提弱、改善弱语音信噪比；**调过头会产生互调、展宽带宽** |
| **AT / TUNE** | 自动天调 | 检测 SWR、自动匹配，维持最低驻波比 |
| **MODE** | —— | 切换 FM / LSB / USB / CW 等工作方式 |

> ⚠️ **发数据（AFSK/FSK/PSK）时要关掉 ALC 和语音压缩**——它们会让信号包络畸变、破坏基带特征、抬高误码率。

### 3.1.4 选呼/静噪编码

- **CTCSS（亚音调静噪）**：在话音上叠加一个 **67–250.3 Hz** 的低频音（人耳几乎听不到），共 38 个，对方收到正确亚音才打开静噪。
- **DCS（数字亚音）**：CTCSS 的数字版，作用相同。
- **DTMF（双音多频）**：8 个音调中取 2 个组合，代表 16 种状态，用于**遥控、拨号**等（不是静噪）。
- **NFM / WFM**：窄带调频（通信，信道 12.5/25 kHz）/ 宽带调频（广播，约 180 kHz）。对应"半频偏 2.5 kHz / 全频偏 5 kHz"。

### 📝 3.1 真题精讲

**真题（单选）** 收发信机面板的 ATT 代表什么？✅ A. 收信机输入衰减器，接收大信号时接入，使信号不过大而前级过载
> ATT 是"给输入信号踩刹车"，专治强信号导致的过载/互调。别和 AGC（自动稳增益）、PRE（放大弱信号）混。

**真题（单选）** 既然有 AGC，为什么还要 ATT 开关？✅ A. 特强带外信号会阻塞前级、产生失真和互调，需在前端加衰减器并用开关控制
> AGC 调的是中频级增益，挡不住**前端**被特强信号阻塞；ATT 在最前端先衰减，二者互补。

## 3.2 天线：增益、类型与极化

> 天线干两件事：发射时**把馈线里的射频电流变成空间电磁波**，接收时**把电磁波变回电流**。注意——天线**不放大功率**，它的"增益"是靠**把能量集中到某个方向**得来的。

### 3.2.1 天线增益与基准（dBi / dBd 必须分清）

天线增益 = **在最大辐射方向上，与基准天线相比信号增强了多少**。基准不同，单位不同：

| 单位 | 基准天线 | 说明 |
|---|---|---|
| **dBi** | **理想点源天线**（各向同性、向所有方向均匀辐射的假想天线） | 理论基准 |
| **dBd** | **半波长偶极天线** | 实用基准 |
| **dB**（不带 i/d） | **未指明基准** | **缺乏参考价值**（商家常玩的文字游戏） |

**最重要的换算关系**：

$$\boxed{\text{dBi} = \text{dBd} + 2.15}$$

因为**半波偶极天线本身相对点源就有 2.15 dBi 的增益**。所以：半波偶极 = **0 dBd = 2.15 dBi**。

### 3.2.2 用 dB 比较天线（结合 S 表）

记牢两个 dB 锚点（详见[第 4 章](./04-电工电子技术基础.md)）：**+3 dB = 功率 2 倍，+6 dB = 4 倍**；**S 表每升 1 格 = 6 dB**。

> 📝 **真题**：甲天线 6.15 dBi，乙天线 1 dBd，同功率同条件，远处信号功率关系？
> **解**：把单位统一——乙 1 dBd = 1 + 2.15 = **3.15 dBi**。甲 6.15 dBi 比乙高 **3 dB**，3 dB = 2 倍 → **甲是乙的 2 倍**。✅

> 📝 **真题**：用半波偶极时对方收到 S4，改用 8.15 dBi 八木（功率不变），对方收到？
> **解**：八木 8.15 dBi 相对半波偶极（2.15 dBi）高 **6 dB** = **1 个 S 格** → S4 + 1 = **S5**。✅
> 若**双方都**换成该八木，则发端 +6 dB、收端也 +6 dB，共 +12 dB = 2 格 → **S6**。（两端增益都算！）

### 3.2.3 常见天线类型

- **半波偶极（Dipole, DP）**：最基础，中点馈电，水平架设时**水平极化**，水平面方向图呈"8"字（两侧强、两端弱）。增益基准 0 dBd。
- **垂直接地天线（GP, Ground Plane）**：1/4 波长垂直振子 + 地网（"镜像地平面"）。**垂直极化**，**水平面全向**（所以叫全向天线），辐射仰角略大于 0°，适合手台/车台。
  - 需要**足够大的导电地平面**做"镜像"，否则谐振频率和阻抗都跑偏——所以车台要把馈线地端接到车身大金属面。
  - 场地受限可**串入电感"加感"缩短**振子；**加感位置越靠顶部，效率越高**（顶 > 中 > 底）。
- **八木（Yagi, BEAM）**：有源振子 + 反射器 + 引向器，**定向、高增益**，用于远距/卫星。
- **手台"橡胶天线"**：是缩短的加感天线，**增益比全尺寸天线低**；手持姿态下垂直极化分量较强；在**车内**用会被**车体屏蔽**而信号变弱。

### 3.2.4 极化匹配（高频考点，规律很简单）

**收发两端天线的极化要一致，信号才最强。** 错配会损失：

| 发射极化 | 接收极化 | 效果 |
|---|---|---|
| 垂直 ↔ 垂直 / 水平 ↔ 水平 | 一致 | **最佳** |
| 垂直 ↔ 水平（正交） | 错配 90° | **最差**（损失很大） |
| 左旋圆 ↔ 左旋圆 / 右旋 ↔ 右旋 | 同旋向 | **最佳** |
| 左旋圆 ↔ 右旋圆 | 反旋向 | **最差** |
| 圆极化 ↔ 线极化（垂直/水平） | —— | **固定损失约 3 dB，但不随姿态变化**（所以"改成另一种线极化，效果不变"） |

**记忆**：**线极化要"同向"，圆极化要"同旋"**；圆配线永远差 3 dB 但稳定——这正是卫星下行常用圆极化的原因：卫星在翻滚，线极化会忽强忽弱，圆极化接收能**避免极化引起的中断**。

> ⚠️ **圆极化"对镜反转"**：右旋圆极化波经反射或换个观察方向会变左旋。题目说"地面收到顺时针旋转的电场"，那是**左旋**（因为沿传播方向看的定义）——这类题按定义判断即可。

## 3.3 馈线与驻波比（SWR）

### 3.3.1 同轴电缆与特性阻抗

- 业余电台几乎都用**同轴电缆**做馈线，**特性阻抗 50 Ω**（这是发射设备的统一标准；75 Ω 多用于电视接收）。
- 选同轴电缆，最关键看两个电气参数：**①特性阻抗 ②指定频率下每百米的损耗**（不是芯线截面积/耐压）。
- 同轴电缆为何常用：**易用、与周围物体互耦低**（不是"损耗最低/功率最大/最便宜"）。

馈线损耗会带来两个坏处：**发信功率降低、收信信噪比下降**。损耗随**频率升高而增大**；**空气介质**电缆损耗最低（但要防潮）；**受潮**会使介质损耗增大、屏蔽层/芯线氧化甚至断路——所以室外电缆**外皮要耐紫外线**（护套坏了就进水）。

**连接器**：**400 MHz 以上优先用 N 型**连接器；M 型（即 PL-259/SO-239）广泛用于 HF/VHF。

### 3.3.2 驻波比（SWR）——理解"反射"

**SWR（电压驻波比）衡量"负载阻抗与馈线特性阻抗的匹配程度"。**

- 当天线阻抗 = 馈线特性阻抗（50 Ω）时**完美匹配**，能量全部送出，馈线上**无驻波，SWR = 1:1**。
- 不匹配时，**部分能量被反射**沿馈线返回，与入射波叠加形成"驻波"——馈线上各处电压/电流出现周期性的最大/最小值。**SWR = 最大电压/最小电压 = 最大电流/最小电流。**
- SWR 越高，反射越多、损耗越大、功放越危险。所以**发射机在 SWR 过高时会自动降功率，保护末级功率半导体器件**。一般要求 **SWR ≤ 1.5（优）/ ≤ 2.0（可用）**。

**为什么家里电线看不到驻波？** 因为市电频率太低（50 Hz，波长 6000 km），导线长度与波长相比微不足道，驻波现象不明显——这道题在考你"驻波是高频长线才显著的现象"。

**SWR 读数不稳定**的原因：①某处**接触不良**；②用 **SSB** 测量时话音幅度本身在变（**改用 CW 或按住 PTT 发连续载波测更准**）。

### 📝 3.3 真题精讲

**真题（单选）** SWR 表显示 4:1 意味着：✅ A. 从发射机输出端看，天线系统整体匹配不佳
> SWR 是匹配程度指标，4:1 = 匹配差、反射大。它**不直接等于**辐射效率或增益，也不能简单换算成"200Ω 或 12.5Ω"（那需要更多信息）。

**真题（单选）** 用同轴电缆接天线，为什么 SWR 趋于 1:1 为好？✅ A. 降低电缆损耗，让射频能量更有效传输
> 匹配好→反射少→驻波小→损耗低→能量高效送到天线。这是 SWR 追求 1:1 的根本原因。

## 3.4 电波传播（A 类重在 VHF/UHF 视距）

### 3.4.1 视距传播与多径

A 类工作的 VHF/UHF 主要是**视距传播**（沿直线、受地形遮挡）：

- **视距距离极限主要取决于**：**收发天线离地面的相对高度**（架得越高、看得越远）——所以中继台都建在高处。
- **多径效应**：电波经直射 + 地面/建筑反射等**多条路径**到达，相互**干涉**（相长或相消）。这就解释了一个常见现象：**在 VHF/UHF 上信号忽强忽弱、移动几步或改变几十 kHz 就明显改善**——因为改变了多径的相位关系。多径还会使数据通信**误码率上升**、ATV 图像出现**重影**。

### 3.4.2 超视距传播（偶发但常考）

有时 VHF/UHF 能传到几百上千公里，靠的是特殊机制：

| 现象 | 成因 | 典型 |
|---|---|---|
| **大气波导** | 对流层**高空逆温** | VHF/UHF 超视距 |
| **对流层散射** | 对流层不均匀散射 | 可达约 **500 km** |
| **突发 E 层（Es）** | 电离层 E 层突现强电离区**反射** | **6 m / 2 m** 上千公里 |
| **流星余迹散射** | 流星电离余迹反射 | **6 m 最适合**，常用 FT4 |
| **EME（月面反射）** | 信号打到月球再反射回地球 | 极弱信号，月球**近地点**最佳 |

> 对比记忆：**短波（HF）靠电离层天波反射**做全球 DX，这是 HF 与 VHF/UHF 最大的不同——但 **A 类没有 HF 权限**，这些属于升级 B/C 后的玩法，了解即可。

## 3.5 收发信机原理与指标

### 3.5.1 调制与解调

- **调制器**：用原始信号**控制载波的幅度/频率/相位**（对应 AM/FM/PM）。
- **解调器**：从已调信号中**还原原始信息**，目标是**尽量忠实还原**。两个专有名词：
  - **检波** = 对 **AM**（调幅）解调（用检波器）；
  - **鉴频** = 对 **FM**（调频）解调（用鉴频器）。

> 📝 FM 接收的一个反直觉点：**不能单凭声音大小判断 FM 信号强弱**！因为 FM 解调出的音量只取决于**频偏**，与信号幅度无关，而且中频信号进鉴频器前已被**限幅**切齐了。（AM/SSB 才是越强越响。）

### 3.5.2 超外差与镜像频率干扰

主流接收机是**超外差式**：用**混频器**把收到的射频信号**变频**到固定的**中频（IF）**再处理。

- **本振 ± 中频** 处会有一个"镜像频率"也能混进中频造成干扰：**镜频 = 信号频率 ± 2×中频**。
- **抑制镜频**靠**变频级之前的预选（波段）滤波器**；**抑制邻频干扰**靠**变频之后的中频滤波器**。

> 📝 **镜频计算**：第一中频 45.05 MHz，收 145.00 MHz，镜频 = 145 ± 2×45.05 = **235.10 或 54.90 MHz**。✅（口诀：信号频率加减两倍中频。）

### 3.5.3 发射机效率与功率

- **发射机效率 = 输出到天线的功率 ÷ 消耗的电源功率**。效率总小于 1，**损耗的能量绝大部分变成热**（小部分成杂散）。
- **电流估算**：由 P电源 = P射频/效率、I = P电源/U。例：13.8 V、效率 80%、射频 N 瓦 → I = N/(13.8×0.8) ≈ **0.091×N** 安。
- **载波功率**：无调制时一个射频周期内供给馈线的平均功率。
- 同样峰值功率下、无语音输入时，实际射频输出功率排序 **FM > AM > SSB**（FM 始终满载波，SSB 无语音时几乎不发）。

### 3.5.4 接收机四大选择性指标（对应不同干扰）

| 指标 | 抗的是哪种干扰 |
|---|---|
| **中频选择性 / 信道选择性** | **邻近频率**干扰（中频滤波器） |
| **前端带宽** | 相距较远的**强**干扰 |
| **镜像抑制比** | 相距 **2 倍中频**处的镜频干扰 |
| **带内波动** | （描述 SSB 频响均匀度，不是抗干扰） |

### 3.5.5 灵敏度（计算题集中地）

**灵敏度 = 达到规定输出质量时所需的最小输入信号。数值越小 = 越灵敏（能收更弱的信号）。**

- 常用单位：功率 **dBm**、电压 **μV / dBμV**。
- 必记锚点：**1 μV（50 Ω 端电压）= −107 dBm = 0 dBμV**；**电压翻倍 = +6 dB**。
  - 0.5 μV → −113 dBm；2 μV（端电压）→ −101 dBm。
- ⚠️ **电动势 vs 端电压的 2 倍关系**：灵敏度若以**电动势（EMF，开路电压）**给出，则它是 50 Ω 上**实际端电压的 2 倍**。所以题目"灵敏度 1 μV（电动势）"对应端电压 0.5 μV = **−113 dBm**；"灵敏度 2 μV（电动势）"对应端电压 1 μV = **−107 dBm**。换算时先判断给的是端电压还是电动势。
- 比较灵敏度要**同质量标准**才有意义：只给"0.1 μV vs 0.15 μV"而不说输出质量标准，**无法直接比较**。
- **制约灵敏度的根本因素是机内噪声**（不是增益）。

### 3.5.6 噪声系数 / 噪声指数 / 噪声温度（弱信号通信指标）

描述设备自身噪声大小，三种等价表述：

| 指标 | 形式 | 理想值 | 越好 | 关系示例 |
|---|---|---|---|---|
| **Fn 噪声系数** | 比值（输入信噪比/输出信噪比） | **1** | 越接近 1 | Fn=2 |
| **NF 噪声指数** | dB（Fn 取对数） | **0 dB** | 越接近 0 | NF=3 dB |
| **Te 噪声温度** | 开尔文 | **0 K** | 越低 | Te=290 K |

记住两组对照：**理想 = (1, 0 dB, 0 K)**；**Fn=2 ↔ NF=3 dB ↔ Te=290 K（17 ℃ 基准）**。**SNR（信噪比）= 信号功率/噪声功率。**

## 📌 本章记忆卡片

- **最小系统**：收 = 天线+解调器；发 = 振荡器+调制器+天线。电源线粗短=减小压降。
- **单工(PTT对讲)/双工(卫星同时)/半双工(中继)**。
- **面板**：SQL 静噪、AGC 稳增益、ATT 衰减强信号、PRE 放大弱信号、NB 切脉冲；PTT 发射、VOX 声控、ALC 防过驱、PROC 语音压缩、AT/TUNE 天调。发数据关 ALC 和压缩。
- **增益**：dBi=dBd+2.15；半波偶极=0 dBd=2.15 dBi；裸 dB 无意义。**+3 dB=2 倍，+6 dB=4 倍=1 个 S 格**。
- **极化**：线极化同向、圆极化同旋为最佳；圆配线固定差 3 dB 但稳定（卫星用圆极化）。
- **馈线**：50 Ω 同轴；选购看特性阻抗+损耗；频率越高损耗越大；400 MHz 以上用 N 型头。
- **SWR**：匹配程度，1:1 最佳；=最大/最小电压比；过高发射机降功率护功放；SSB 测不稳→用 CW 测。
- **传播**：VHF/UHF 视距，极限取决于天线相对高度；忽强忽弱=多径；超视距=大气波导/对流层散射/突发 E 层（6m/2m）。
- **原理**：检波=解 AM、鉴频=解 FM；超外差镜频=信号频率±2×中频，预选滤波器抑镜频、中频滤波器抑邻频。
- **灵敏度**：越小越灵敏；1 μV@50Ω=−107 dBm=0 dBμV，电压翻倍+6 dB；电动势=端电压×2；制约因素=机内噪声。
- **噪声**：理想 (Fn=1, NF=0 dB, Te=0 K)；Fn=2↔NF=3 dB↔Te=290 K。
